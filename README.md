# 风暴钟塔 · Storm Clocktower

一款 2D 纵向蓄力跳跃平台游戏：暴风雨夜，小灯守「灯芯」要在黎明前攀上废弃钟塔顶端，敲响晨钟。核心操作为「按住蓄力、松开跳跃、失败后向下坠落」，全部角色、地图、美术与音效均为程序化原创（Canvas 绘制 + WebAudio 合成，无任何外部素材）。

## 运行方式

```bash
npm install
npm run dev      # 开发模式（默认 http://localhost:5173）
npm run build    # 类型检查 + 生产构建（输出 dist/）
npm run preview  # 预览构建产物
npm test         # 运行单元测试（vitest，36 例）
```

技术栈：HTML5 Canvas + TypeScript + Vite。不使用游戏引擎与第三方物理引擎，物理与碰撞全部手写。

## 操作说明

| 按键 | 作用 |
| --- | --- |
| A / D 或 ← / → | 地面移动；空中有限方向修正（每跳有速度预算） |
| 按住 空格 | 蓄力（角色压缩、灯焰变亮、头顶出现蓄力条） |
| 松开 空格 | 起跳；蓄力时间决定垂直高度与水平距离，蓄满自动起跳 |
| R | 快速重开一局 |
| Esc | 暂停 / 继续 |

空中撞墙会反弹，从高处坠落会记入「坠落次数」；掉出世界会自动回到最近点亮的检查点（坠落 +1，碎裂平台复原，保证不存在不可恢复状态）。

## 关卡与机制

- **5 个连续向上的区域**：坍塌基座 → 齿轮库 → 结冰钟廊 → 风暴外壁 → 钟顶，摄像机平滑跟随。
- **5 类平台**：普通石台、单向木板（可从下方穿过）、移动平台（可载人、带轨道提示）、碎裂平台（踩上约 0.45s 后崩塌、3.2s 后复原）、铜制弹簧（高弹）。
- **环境机制（2 种）**：**横风**（高区阵风摆动，仅作用于空中，有 HUD 风向指示与流线特效）与**结冰地面**（低摩擦打滑，带冰凌高光）。
- **10 个齿轮收集物**、**3 个检查点灯笼**（点亮后成为重生点，但保留坠落计数惩罚）、山顶金钟为明确终点。
- 结算界面统计**用时、坠落次数、最高高度、收集数量**；最佳成绩（以用时为准）通过 `localStorage` 保存并在刷新纪录时提示「新纪录」。

## 视觉与反馈

- 三层以上视差背景：远景风暴云 + 月亮、中景废墟塔群剪影、近景悬链/断梁/旋转巨齿轮，另有全屏斜向雨幕与随机闪电。
- 原创角色「灯芯」：披风、兜帽、发光双眼、随风围巾、腰间提灯，具备待机呼吸、行走摆腿、蓄力压缩、跳跃拉伸、落地压扁、眨眼等动画。
- 蓄力火花、起跳/落地尘土、收集爆光、检查点光环、通关金光与钟声（WebAudio 程序化音效）逐一反馈。
- 逻辑分辨率固定 960×540，窗口缩放时等比适配并居中，画面不会错位。
- 首次进入显示标题操作说明，开局附带「首次攀登提示」浮层（完成首跳后消失并记忆）。

## 代码结构

```
src/
  main.ts             入口：固定 120Hz 步长主循环、DPR/缩放适配
  engine/physics.ts   纯物理：蓄力、跳跃速度、AABB 分轴碰撞（可测试）
  game/level.ts       关卡数据：平台/风区/收集物/检查点/终点
  game/session.ts     一局状态：检查点、坠落、收集、高度（可测试）
  game/storage.ts     最佳成绩 localStorage 持久化（可注入、可测试）
  game/game.ts        主逻辑状态机：玩家、平台机制、风、摄像机
  game/render.ts      渲染：视差背景、平台、角色、HUD、界面
  game/input.ts       键盘输入轮询
  game/audio.ts       WebAudio 程序化音效
  game/particles.ts   粒子系统
tests/                charge / collision / checkpoint / storage 单元测试
```

## 测试

`npm test` 覆盖四组关键逻辑，共 36 例，全部通过：

- `charge.test.ts`：蓄力累积/封顶、最低蓄力比、跳跃速度映射与夹紧；
- `collision.test.ts`：落地贴合、左右撞墙、顶头、单向平台三种情形、持续贴地；
- `checkpoint.test.ts`：检查点激活/去重/重生点更新、坠落统计、收集去重、关卡要素数量；
- `storage.test.ts`：首次保存、更优覆盖、更差保留、损坏 JSON 自动恢复。
